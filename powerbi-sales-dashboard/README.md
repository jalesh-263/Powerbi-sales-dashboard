Power BI Sales Dashboard

This project showcases an interactive Power BI dashboard created to analyze sales data and generate business insights.

Objective
To visualize sales performance, identify trends, and track key performance indicators (KPIs) using Power BI.

Tools Used
- Power BI
- Excel / CSV dataset

Key Insights
- Overall sales trend analysis
- Region-wise and category-wise performance
- Identification of top-performing products
- KPI metrics such as total sales, profit, and quantity

Dashboard Features
- Interactive slicers and filters
- Dynamic charts and visuals
- Clean and user-friendly layout
